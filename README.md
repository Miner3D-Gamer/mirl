# **[Mirl](.markdown/https://github.com/Miner3D-Gamer/mirl)**

### A mini ecosystem of crates that all follow the [Mirl Guidelines](.markdown/guidelines.md) and try to work well with one another.

> Disclaimer:
>
> 1.  This mini ecosystem is still a WIP due to the recent split/reorganization.
> 2.  Most of these libs require nightly to access the `feature` keyword (for faster runtime execution, `no_std` support, and better customizability.)

The crates contained here vary in purpose so it is better to better to inspect their descriptions individually [here](.markdown/crates.md#these-aspects-are-now-offered-in-specialized-packages)
